# AKPLAY – Stream Your World

A modern streaming platform built with React, TypeScript, and Vite.

</div>

## About

AKPLAY is a next-generation streaming platform featuring a sleek glassmorphism UI, an AI-powered chatbot, a fully functional admin CMS panel, and a Netflix-style intro animation.

## Features

- **Netflix-style Intro Animation** – Custom logo animation with ta-dum sound on launch
- **Apple Glass Finish UI** – Glassmorphism design with mesh gradient backgrounds
- **Admin CMS Panel** – Full content management system with login authentication
  - Manage videos, text content, and community posts
  - Changes reflect across the entire website in real-time
  - "Publish to Codebase" persists changes to the project files
- **AI Chatbot** – Powered by Google Gemini 2.5 Flash
- **Responsive Design** – Optimized for all screen sizes
- **Collapsible Sidebar** – Clean navigation with toggle support
- **Video Protection** – Right-click and screenshot prevention on video content
- **Subscription Plans** – Three-tier pricing page

## Tech Stack

- **Frontend:** React 19, TypeScript 5.8
- **Build Tool:** Vite 6
- **Styling:** Tailwind CSS 4, custom glassmorphism utilities
- **Animation:** Motion (Framer Motion)
- **Icons:** Lucide React
- **AI:** Google Generative AI (Gemini 2.5 Flash)
- **Routing:** React Router Dom 7

## Getting Started

**Prerequisites:** Node.js

1. Clone the repository:
   ```bash
   git clone https://github.com/samarthrao34/akplay.git
   cd akplay
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file and add your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

4. Run the dev server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000)
